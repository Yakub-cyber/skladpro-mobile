# Политика конфиденциальности СкладПро

_Последнее обновление: 2026-07-14_

Приложение «СкладПро» (далее — «Приложение») предназначено для учёта товаров, продаж и доставки внутри компании пользователя.

> **Не юридическая консультация.** Этот шаблон закрывает базовые требования Google Play. Перед публикацией дай его проверить юристу, добавь свои реквизиты и разместись по постоянному URL (например, GitHub Pages или собственный сайт). URL укажи в Play Console → Store listing → Privacy policy.

## 2. Какие данные мы собираем

| Категория | Пример | Цель | Хранение |
|---|---|---|---|
| Учётные данные | email, хешированный пароль | Вход в аккаунт | Supabase Auth |
| Данные компании | название компании, товары, заказы, клиенты, поставщики, сотрудники, документы | Работа приложения | Supabase (Postgres, ЕС/US-регион) |
| Токен push-уведомлений | FCM device token | Отправка уведомлений о заказах | Supabase, поле `push_tokens` |
| Диагностика | сообщения об ошибках, stack trace | Отладка (только когда подключён Sentry) | Не собирается по умолчанию |

Приложение **не собирает**: точное местоположение, звук, контакты, историю просмотров, рекламные идентификаторы.

## 3. Разрешения

- **Камера** — сканирование штрихкодов товаров при приёмке/продаже.
- **Уведомления** — push о новых и назначенных заказах.
- **Интернет** — синхронизация с сервером.

Разрешения `RECORD_AUDIO`, `SYSTEM_ALERT_WINDOW`, `READ_CONTACTS`, `ACCESS_FINE_LOCATION` не запрашиваются.

## 4. Кому передаём

- **Supabase Inc.** — облачное хранение данных компании и авторизация. [Privacy policy Supabase](https://supabase.com/privacy).
- **Google Firebase Cloud Messaging** — доставка push-уведомлений. [Google Privacy Policy](https://policies.google.com/privacy).

Данные не продаются третьим лицам и не используются для рекламы.

## 5. Хранение и удаление

Данные компании хранятся, пока действует подписка. По запросу владельца компании данные удаляются в течение 30 дней.

**Запросить удаление аккаунта:** отправь письмо на _(email из п.1)_ с темой «Удаление данных».

## 6. Права пользователя

Ты имеешь право:
- получить копию своих данных;
- потребовать удаление;
- отозвать согласие на обработку.

Запросы — на email в п.1.

Мы можем обновить эту политику. Новая версия публикуется по этому же URL с новой датой.
